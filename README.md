# AI-Based Spam and Abuse Detection System

This project implements an AI-powered moderation system for online comment platforms. The system detects spam and abusive content using AI analysis and provides moderation tools for administrators.

## Features
- User comment posting
- AI spam and abuse detection
- Moderator dashboard
- Admin management system
